An interactive visual laboratory for learning **Data Communications** through simulation and visualization.

The goal is not only to generate waveforms, but to help students understand **why each communication technique exists, what problem it solves, and what trade-off it introduces**.

## 🚀 Features

### 🔢 Line Coding Laboratory

Digital Data → Digital Signal

Supported:

- Unipolar NRZ
- Polar NRZ-L
- Polar NRZ-I
- Manchester
- Differential Manchester
- Bipolar AMI

For each technique:

- Concept explanation
- Encoding rule
- Mathematical model
- Signal visualization
- Advantages & disadvantages
- Real-world applications

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- KaTeX

## 🧠 Learning Approach

Instead of memorizing formulas:

Problem → Idea → Rule → Signal → Trade-off
